# MLB Home Ticket Tracker

Single-user self-hosted service that tracks the cheapest current ticket price for upcoming MLB home games and publishes each game's price into Home Assistant through MQTT discovery.

This repo is optimized for one person running one small service on their own server. The first working source is Ticketmaster using the public Discovery API.

## Status

- Ticketmaster: planned first-class source
- SeatGeek: planned partial adapter, disabled by default
- Vivid Seats: planned unsupported/experimental scaffold, disabled by default

Full setup and configuration instructions will live here as implementation lands.
